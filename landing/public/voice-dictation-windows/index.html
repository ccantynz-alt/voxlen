FAQ

Frequently asked questions

Everything you need to know about Voxlen for Windows.

Is Voxlen a good Dragon NaturallySpeaking alternative for Windows?
Yes. Dragon NaturallySpeaking (Dragon Professional) still exists on Windows but is expensive ($300–$500+), relies on aging acoustic models, and lacks modern AI features. Voxlen replaces it with state-of-the-art AI transcription (Deepgram Nova-2) and AI grammar correction (Claude / GPT-4o-mini) at a fraction of the price, with active development and a free tier to get started.
Does Voxlen work on Windows 11?
Yes. Voxlen is fully compatible with Windows 11 (Home, Pro, Enterprise, Education — all editions) and Windows 10 (version 1903 and later). It is a native Windows application, so it installs and behaves like any standard Windows program with no unusual dependencies.
Which Windows apps does Voxlen work in?
Voxlen works in virtually any Windows application — Microsoft Word, Outlook, Excel, Microsoft Teams, Edge, Chrome, Firefox, Notepad, legal practice management platforms, QuickBooks, Xero, and more. It injects text at the OS level using the Windows Input Simulation API, so anywhere you can type, Voxlen can dictate.
Is my dictation data private on Windows?
Yes. Voxlen is designed for professionals such as lawyers and accountants who handle sensitive, confidential information. Your dictated content is never uploaded to or stored on Voxlen servers. All vocabulary learning and correction patterns are stored locally on your Windows PC. Privacy is a core architectural guarantee, not an afterthought.
Does Voxlen work with Windows Speech Recognition or Cortana?
Voxlen is a completely independent application — it does not rely on Windows Speech Recognition, Cortana, or any built-in Windows AI features. It brings its own AI engines (Deepgram Nova-2 and OpenAI Whisper) which deliver far superior accuracy to anything built into Windows, especially for professional terminology.
